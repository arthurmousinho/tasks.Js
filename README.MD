# Tasks.Js
Lista de tarefas com armazenamento em Local Storage e timer para aplicar a técnica Pomodoro.

![image1](./assets/github/image1.png)
![image2](./assets/github/image2.png)

## ⚙️ Funcionalidades
- Adicionar tarefas
- Armazenar as tarefas no Local Storage do navegador
- Concluir/Remover as tarefas criadas
- Timer de 25 minutos para aplicar a técnica Pomodoro

## 🛠️ Stack utilizada
- Html
- Css
- JavaScript

## 📚 Aprendizados
- Entender melhor sobre a DOM e organização das ideias usando a sintaxe da linguagem JavaScript.
- Reforçar os conhecimentos sobre a construção de layouts com HTML e CSS.
- Melhorar a habilidade de pesquisar sobre tópicos em caso de dúvida.
